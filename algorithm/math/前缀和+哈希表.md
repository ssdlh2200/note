# 前缀和+哈希表
- leetcode 560 和为 K 的子数组
- leetcode 974 和可被 K 整除的子数组

题目的核心在于找出满足条件的连续子数组，那么我们就能用到前缀和这个工具
先来看下前缀和
![[20251119-12-53-46.png]]
- 设 i < j，那么 s\[j\] - s\[i\] 表示的就是 (i,j]之间的元素和

**和为K的子数组**
- 当 k = s\[j\] - s\[i\] 时，说明该子数组满足和为K
那么公式就可以转换为
- s\[j\] - k = s\[i\]
此时我们只需要知道s\[i\]出现的次数，那么我们就可以通过hashmap来完成这个统计

**和可被K整除的子数组**
- 当 (s\[j\] - s\[i\]) mod k = 0 时，说明该子数组满足能够被K整除 
- s\[j\] mod k - s\[i\] mod k = 0
- s\[j\] mod k = s\[i\] mod k
注意：c语言中的取余不是数学意义上的取余，可以用(a mod k + k) mod k实现数学取余
